# Resonance : My Personal Music Catalog
Resonance is a progressive web application (PWA) that displays my personal music taste through data from Spotify's API.


## Features
- Displays liked songs, top artists, top albums and playlists from Spotify
- Allows users to discover a random liked song daily
- Toggle for dark and light mode
- Application is responsive and works on mobile, tablet and desktop

## Tech Stack
- Frontend: React, Vite
- Backend: Node.js, Express
- Database: PostgreSQL
- Deployment: Railway
